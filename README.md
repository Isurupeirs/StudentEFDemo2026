# Student Management System

A WPF database-driven application built with C# and Entity Framework.

## Features
- Insert Students
- Load/View All Students
- Search Students by Name
- Sort Students Alphabetically
- Count Students
- Get First Student
- Update Student
- Delete Student

## Technologies Used
- C# .NET Framework
- WPF (Windows Presentation Foundation)
- Entity Framework (Code First)
- LINQ Queries
- SQL Server LocalDB

## How to Run
1. Clone the repository
2. Open in Visual Studio
3. Install EntityFramework via NuGet
4. Run the application
